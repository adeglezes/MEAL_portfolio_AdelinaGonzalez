# Mixed-Methods Evaluation Examples

This folder contains examples of mixed-methods approaches commonly used in Monitoring, Evaluation and Learning (MEL).

The documents illustrate:
- Integration of qualitative, quantitative and administrative data.
- Triangulation to strengthen credibility and depth of analysis.
- Outcome harvesting steps in a participatory context.
- Development of contribution narratives based on evidence.

All examples use simulated or generic content.
