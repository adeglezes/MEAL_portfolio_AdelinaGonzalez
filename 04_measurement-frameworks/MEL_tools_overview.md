# MEL Tools Overview

This document summarizes examples of tools used in MEL systems.  
These examples reflect common practices across multi-country or multi-partner programs.

## Data Collection Tools
- Mobile data collection platforms (ODK/GeoODK-style tools).  
- Structured surveys and short tests.  
- Field logbooks or monitoring templates used by implementing partners.  
- Observation checklists.  

## Data Management and Monitoring
- Excel- or database-based indicator tracking systems.  
- Dashboards for visualizing progress across locations or indicators.  
- Version-controlled scripts for cleaning and analysing monitoring data.  

## Evaluation and Learning Tools
- Theory of Change visualizations.  
- Outcome harvesting templates.  
- Reflection session protocols.  
- Case study templates.  
- Contribution analysis frameworks.  

## Reporting Tools
- Structured reporting formats for donor submissions.  
- Visualization tools (R, Python, spreadsheets) for summarizing findings.
