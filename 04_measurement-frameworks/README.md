# Measurement Frameworks

This folder contains examples of measurement tools and frameworks used in Monitoring, Evaluation and Learning (MEL).  
The materials illustrate:

- A generic Theory of Change (ToC) structure.  
- An indicator matrix template adapted for MEL systems.  
- Examples of MEL tools used in implementation or evaluation.  
- A concise reference on core measurement concepts, including reliability, validity and CTT/MTMM logic.  

All content is simulated or generic to maintain confidentiality.
