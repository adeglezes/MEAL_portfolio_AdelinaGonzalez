# Monthly Social Media Metrics Report (Template)

## 1. Overview
Summary of the content period analysed.

## 2. Key Numbers
- Total reach
- Impressions
- Engagement and engagement rate
- Follower growth
- Top-performing content
- Platform comparison (e.g., IG, FB, TikTok)

## 3. Trends
- Increases or decreases in interaction
- Format-level performance (e.g., reels vs. static posts)
- Posting frequency and its effect

## 4. Insights
- What resonated with the audience?
- What content led to saves or shares?
- What days/times performed best?

## 5. Recommendations
- Optimal posting patterns
- Topics that generate engagement
- Content types to prioritize
- Suggestions for testing A/B variations

## 6. MEL Link
For IWPR-style communication MEL:
- Which audiences are engaging?
- Which topics are gaining traction?
- Are messages reaching key groups?
- Are online discussions aligned with program learning?

This reporting format can be adapted for internal learning sessions.
