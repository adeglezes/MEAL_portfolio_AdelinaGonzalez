# Social Media Analytics (Metricool)

This folder includes examples of how social media metrics can be collected, organised and interpreted for MEL or communication-related analysis. The examples are based on anonymised or simulated structures reflecting typical Metricool outputs.

Contents:
- Overview of key platform metrics and relevance for MEL.
- An evaluation-style interpretation of social media insights.
- A template for reporting engagement, reach, trends and content performance.
- A simple Python script to parse a CSV of social media metrics.
