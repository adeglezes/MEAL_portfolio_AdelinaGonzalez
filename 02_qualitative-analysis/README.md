# Qualitative Analysis Examples

This folder contains qualitative analysis examples relevant to Monitoring, Evaluation and Learning (MEL).  
The materials illustrate:

- Coding frameworks used for thematic analysis.  
- Interview and qualitative data collection procedures.  
- Participatory MEL approaches such as outcome harvesting.  
- Sensemaking techniques for collaborative interpretation of findings.  

All content is illustrative and based on simulated, anonymized examples.
