# Risk Assessment Template for MEL Activities

This template supports planning and mitigation when collecting data in sensitive or restrictive contexts.

## 1. Context Risks
- Political sensitivity  
- Stigma or discrimination  
- Surveillance or monitoring  
- Access constraints  

## 2. Participant Risks
- Reprisal for sharing opinions  
- Exposure of vulnerable groups  
- Emotional distress  

## 3. Data Risks
- Identification through patterns  
- Misuse of information  
- Accidental disclosure  

## 4. Mitigation Measures
- Use anonymous or low-risk data collection methods  
- Limit personally identifiable information  
- Secure data storage and restricted access  
- Clear communication about confidentiality  

## 5. Decision
- Proceed as planned  
- Proceed with modifications  
- Suspend data collection  

This template can be updated based on specific program needs.
